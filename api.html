Getting Started

Quick start guide to using the ExoPlanet AI API

Base URL

https://api.exoplanet-ai.com/v2

Authentication

API key required in header:

Authorization: Bearer YOUR_API_KEY

Rate Limits

  • Free Tier: 100 requests/hour
  • Pro Tier: 1,000 requests/hour
  • Enterprise: Unlimited

Response Format

All responses are in JSON format with consistent structure:

{
  "success": true,
  "data": {...},
  "message": "Success"
}

API Endpoints

Complete reference for all available endpoints

POST
/predict
Analyze exoplanet data and get classification results

Request Body

{
  "orbitalPeriod": 365.25,
  "transitDuration": 4.2,
  "planetaryRadius": 1.0,
  "transitDepth": 0.008,
  "stellarMagnitude": 4.83,
  "equilibriumTemperature": 288
}
const response = await fetch('https://api.exoplanet-ai.com/v2/predict', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_API_KEY'
  },
  body: JSON.stringify({
    orbitalPeriod: 365.25,
    transitDuration: 4.2,
    planetaryRadius: 1.0,
    transitDepth: 0.008,
    stellarMagnitude: 4.83,
    equilibriumTemperature: 288
  })
});

const data = await response.json();
console.log(data);
import requests

url = "https://api.exoplanet-ai.com/v2/predict"
headers = {
    "Content-Type": "application/json",
    "Authorization": "Bearer YOUR_API_KEY"
}
data = {
    "orbitalPeriod": 365.25,
    "transitDuration": 4.2,
    "planetaryRadius": 1.0,
    "transitDepth": 0.008,
    "stellarMagnitude": 4.83,
    "equilibriumTemperature": 288
}

response = requests.post(url, json=data, headers=headers)
result = response.json()
print(result)
curl -X POST https://api.exoplanet-ai.com/v2/predict \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "orbitalPeriod": 365.25,
    "transitDuration": 4.2,
    "planetaryRadius": 1.0,
    "transitDepth": 0.008,
    "stellarMagnitude": 4.83,
    "equilibriumTemperature": 288
  }'

Parameters

Parameter Type Required Description Range
orbitalPeriod number Yes Orbital period in days 0.1 - 10,000
transitDuration number Yes Transit duration in hours 0.1 - 24
planetaryRadius number Yes Planet radius in Earth radii 0.1 - 50
transitDepth number Yes Transit depth percentage 0.001 - 10
stellarMagnitude number Yes Host star magnitude -5 - 20
equilibriumTemperature number Yes Planet temperature in Kelvin 50 - 3000
